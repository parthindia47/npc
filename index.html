What is Fibonacci Network ?

Fibonacci network token ( FCN ) is a token created on binance chain for following people :

  • Game developers and gamers -> token users can play many social games using our token and earn them during the play
  • Content creators -> token users can subscribe to their favorite news site, blogs and streaming channels
  • Artiest -> token users can reward their favorite dancer , painter , or performer the easiest way

How to buy ?

To interact with Fibonacci Network and other decentralized applications on the Binance Smart Chain, you will first need to create a wallet compatible with the Smart Chain network like MetaMask.

  • Setup MetaMask https://metamask.io/
  • Connect to BSC Mainnet
  • Add FBN Contract Address :

    0x713557dd6c2e7307764f479614a8e738e9fccc51

  • Buy via MetaMaskSwap or PancakeSwap
  • Congratulations for joining fiboniacs community!
