What is fibonacci token ?

Fibonacci token is native token of fibonacci network which is currently hosted on binance smart chain.


What is fibonacci wallet ?

Fibonacci wallet is a custodial wallet i.e which supports off chain transactions.Users can use it as any other crypto wallet as well as off chain hot wallet. Fibonacci wallet also only wallet which supports mining of fibo token.
